# Virtu Capital 管理员端操作指南

Virtu Capital 平台的管理员端操作文档，涵盖投资者管理、股票管理、市场数据管理、审核记录、系统设置等全功能操作说明。

## 在线访问

👉 [https://wxmlife.github.io/virtu-capital-admin-guide/](https://wxmlife.github.io/virtu-capital-admin-guide/)

## 本地预览

直接在浏览器打开 `admin-guide/index.html` 即可。

## 技术栈

- 纯 HTML + CSS + JavaScript
- 无外部依赖
- 响应式设计，支持移动端
- 支持深色/浅色主题切换
- 支持打印 / 导出 PDF

## 部署

本项目使用 GitHub Pages 部署。推送到 `main` 分支后，GitHub Actions 会自动部署。
